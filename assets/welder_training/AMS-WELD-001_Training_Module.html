/ Course overview

โมดูลนี้รวบรวมสื่อการอบรมช่างเชื่อมทั้งหมดของหลักสูตร AMS-WELD-001 ไว้ในที่เดียว เพื่อใช้ในระบบ HRM (M Training) และสำหรับการทดสอบการยอมรับ (UAT) แต่ละบทมีบทเรียน แบบทดสอบ และเกมฝึกแบบโต้ตอบ

This module gathers all AMS-WELD-001 welder-training materials in one place for the HRM (M Training) and for UAT. Each chapter has a lesson, an assessment and an interactive trainer.

/ UAT & embedding in the HRM

✓ รายการตรวจ UAT / UAT checklist

  • เปิดบทเรียนได้ครบทั้ง 8 บท each of the 8 lessons opens
  • เปิดแบบทดสอบได้ครบทั้ง 8 บท each of the 8 assessments opens
  • เกมฝึกเล่นได้และคิดคะแนน/แสดงผ่าน-ไม่ผ่าน each trainer plays & scores (pass/fail)
  • บทสรุปผู้บริหารแสดงถูกต้อง the executive summary renders correctly
  • เนื้อหาทวิภาษาและตราบริษัทถูกต้อง bilingual content & branding correct

▸ วิธีรวมเข้าระบบ HRM / How to include

  • วางไฟล์ทั้งหมดไว้ในโฟลเดอร์เดียวกัน แล้วเปิดหน้านี้เป็นหน้าเริ่ม keep all files in one folder; open this page as the index
  • ฝังใน HRM ผ่าน iframe หรือ Streamlit components.html(...) เมื่อให้บริการแบบ http เกมจะฝังแบบ inline ได้ embed via iframe or Streamlit components.html; when served over http the trainers can embed inline
  • ใน M Training: ผูกบทเรียน=วิดีโอ/เอกสาร · เกม=ชนิด interactive · แบบทดสอบ=test (pass 70%) map lesson=video/doc · game=interactive kind · test=test (pass 70%)
